# 商品圖片自動浮水印

本版針對「我的商品」圖片新增自動品牌浮水印：

- 後台新增或更換商品圖片後，系統會自動把「嘎比嘎比孔雀魚」浮水印套到上傳圖片。
- 浮水印採兩層設計：
  - 中央極淡斜角品牌字，降低圖片被直接裁切盜用的機率。
  - 右下角淡淡品牌簽名，讓圖片來源更清楚。
- 浮水印透明度刻意保持低調，避免破壞商品本身的顏色、紋路與觀看體驗。
- 後台商品列表新增 action：「重新套用嘎比嘎比孔雀魚浮水印」，可針對舊商品圖片手動處理。

## 注意事項

1. 未來新上傳或更換的商品圖片會自動套用浮水印。
2. 既有商品圖片若要補套用，請到 Django Admin 的「我的商品」列表勾選商品，使用「重新套用嘎比嘎比孔雀魚浮水印」。
3. 請不要對同一張已經套過浮水印的圖片重複套用太多次，避免浮水印變深。
4. 此版沒有新增資料表欄位，所以不需要新增 migration；但如果你剛剛才導入官方網站內容版，仍需先執行那版原本需要的 `python manage.py migrate`。
