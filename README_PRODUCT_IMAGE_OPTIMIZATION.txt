# 商品圖片壓縮與浮水印優化

本版針對「商品資訊」圖片處理做兩個優化：

1. 浮水印可視度微調
   - 中央斜角「嘎比嘎比孔雀魚」透明度由原本偏淡提升到 0.12。
   - 右下品牌簽名維持低調，透明度微調到 0.24。
   - 目標是讓盜圖防護更明顯，但不破壞孔雀魚顏色、紋路與商品主體觀看體驗。

2. 上傳圖片自動壓縮
   - 商品主圖最長邊會自動縮到 1600px 以內，不會放大小圖。
   - JPEG 會用 progressive + optimize 儲存，品質預設 85。
   - WebP 會以品質 84 儲存。
   - PNG/BMP/GIF 等較容易過大的格式，會轉成 JPEG storefront 圖片，避免媒體檔案太大。
   - 儘量控制在約 950KB 以下；若圖片內容很複雜，會適度降低品質，但避免壓到明顯模糊。

注意：Django ImageField 只在資料庫存圖片路徑，圖片檔案本身存在 media storage。本版主要減少 media 檔案大小與前台載入時間。

這版沒有新增資料表欄位，不需要 migrate。
