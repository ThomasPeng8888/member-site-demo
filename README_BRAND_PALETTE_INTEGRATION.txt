# 嘎比嘎比孔雀魚品牌色系整合版

本版是在「官方 Logo 導覽列 + UIUX 寬版 + QR Code 掃描流程」的基礎上，進一步把網站視覺調整成更符合官方 Logo 的品牌色系。

## 調整重點

1. 以 Logo 深藍作為主品牌色
   - 主要按鈕、導覽列重點、標題重點、連結文字改為深藍 / 藍色系。

2. 降低原本偏綠的比例
   - 原本偏青綠的按鈕與 hover 效果改成深藍到水藍漸層。
   - 青綠保留為少量水族感點綴。

3. 保留水族清透感
   - 背景仍保留淡水藍、霧藍、透明玻璃感與泡泡元素。
   - 右側水族卡片改為更貼近藍色品牌調性。

4. 統一卡片、邊線、陰影
   - 卡片邊線與陰影改成深藍低透明度，整體更正式。
   - 不影響原本 RWD、QR Code、LINE、活動抽獎、核銷與加點功能。

## 修改檔案

- pages/static/pages/css/site.css

本版沒有修改 models，因此不需要 migrate。
