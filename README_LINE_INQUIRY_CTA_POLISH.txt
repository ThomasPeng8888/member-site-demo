GuppyGuppy official website conversion polish
============================================

本版調整重點：

1. 商品詳情頁加入「LINE 詢問此商品」轉換區塊
   - 點擊後會開啟 LINE 官方帳號。
   - 同時嘗試自動複製商品詢問文字，內容包含商品名稱與商品頁連結。
   - 使用者進入 LINE 後可直接貼上詢問，降低詢問門檻。

2. 手機版商品頁加入底部固定 CTA
   - 手機瀏覽商品頁時，底部固定顯示「LINE 詢問此商品」。
   - 方便從 IG / LINE / FB / TikTok 點進來的手機訪客直接詢問。

3. 首頁加入更明確的行銷 CTA
   - Hero 區加入「LINE 詢問商品」。
   - Hero 下方新增 Quick Action 區塊，引導使用者查看商品、加入 LINE 官方、加入會員或參加會員抽獎。

4. 手機版細節微調
   - CTA 按鈕在手機版改為直向滿版，避免擠壓。
   - 商品詳情頁快速詢問區塊縮小間距與字級，提升手機可讀性。

安全性與相容性：
- 不修改資料表。
- 不新增 migration。
- 不影響 Cloudflare R2 商品圖片儲存。
- 不影響首頁純圖片輪播。
- 不影響會員抽獎、活動抽獎、LINE 綁定與後台核銷。

注意：
- LINE 官方帳號目前使用 https://lin.ee/xyjkhCZ。
- 因 LINE 官方帳號短連結不一定支援直接預填文字，所以本版採用「開啟 LINE 官方帳號 + 自動複製商品詢問文字」的穩定做法。
