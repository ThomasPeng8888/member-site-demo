# 商品圖片手機版 / 前台顯示修正

本版修正商品圖片在前台與手機版可能顯示異常的問題。

## 修正內容

1. 將 STATIC_URL / MEDIA_URL 改為絕對路徑 `/static/` 與 `/media/`。
   這可避免在 `/products/`、`/products/<slug>/` 等子路徑頁面中，圖片或靜態檔被瀏覽器誤解析成相對路徑。

2. 商品圖片容器加強為固定正方形顯示。
   商品列表、首頁精選商品、商品詳細頁在手機版都維持 1:1 圖片比例。

3. 商品圖片改用更穩定的絕對定位填滿容器。
   可減少手機瀏覽器或不同版型下圖片高度判斷異常造成的空白問題。

4. 加入 loading="lazy" 與 decoding="async"，提升前台載入體驗。

## 注意

這版沒有改資料表，不需要 migrate。

如果 Render 線上仍看不到商品圖，但本機正常，請確認商品圖片是否真的有在 Render 後台重新上傳。
本機 `media/` 資料夾通常不會跟著 Git 部署到 Render。
