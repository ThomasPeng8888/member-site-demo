# 嘎比嘎比孔雀魚官方網站導覽與內容後台版

本版把網站定位從單純會員中心，升級為「官方網站 + 會員中心」的一站式架構。

## 導覽列層級

桌機版主選單調整為：

1. 首頁
2. 最新消息
3. 我的商品
4. 店鋪資訊
5. 活動專區
6. 會員中心

登入後仍保留會員資料、點數、票券、會員抽獎、我的獎品、LINE 綁定等原功能；店員工具仍只在有權限時顯示。

## 新增後台內容

在 Django Admin `/admin/` 新增三個內容管理項目：

- 最新消息
- 我的商品
- 店鋪資訊

### 最新消息

可設定：標題、分類、摘要、內容、列表圖示、置頂、公開狀態、發布時間。

### 我的商品

可設定：商品名稱、分類、簡介、說明、商品圖片、價格文字、商品狀態、首頁精選、公開狀態、排序。

商品圖片目前支援一張主圖，前台會自動呈現在商品列表、商品詳細頁與首頁精選商品區。

### 店鋪資訊

可設定：店鋪名稱、短標語、店鋪介紹、地址、營業時間、電話、LINE、社群、Google Map、到店提醒、主要店鋪與公開狀態。

## 新增前台頁面

- `/news/` 最新消息列表
- `/news/<slug>/` 最新消息詳細頁
- `/products/` 我的商品列表
- `/products/<slug>/` 商品詳細頁
- `/stores/` 店鋪資訊

首頁也新增最新消息、精選商品、店鋪資訊預覽區。

## 部署注意

本版新增資料表，需要執行 migration。Render 的 Build Command 原本已有：

```bash
pip install -r requirements.txt && python manage.py collectstatic --noinput && python manage.py migrate
```

因此 push 後 Render 會自動建立新資料表。

商品圖片目前使用 Django media upload。若未來商品圖片會大量使用或希望跨部署長期穩定保存，建議再改成雲端物件儲存或 Render Persistent Disk。
