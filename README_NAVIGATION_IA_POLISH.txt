# Navigation IA polish

本版優化導覽列資訊架構，目標是讓網站更像正式一站式官方會員平台。

## 導覽層級

### 首頁
品牌入口與主要介紹。

### 會員中心
集中會員相關功能：
- 會員資料總覽
- LINE 綁定設定
- 會員抽獎
- 會員優惠
- 我的票券
- 我的獎品
- 點數紀錄

未登入會員會看到：
- 免費加入會員
- 會員登入
- 會員抽獎
- 會員優惠

### 活動專區
集中活動相關功能：
- 活動資訊
- 活動抽獎
- 我的活動抽獎（登入後顯示）

### 店員工具
只在具備店員或管理權限時顯示：
- 店員加點
- 抽獎核銷
- 活動獎品核銷
- 活動抽獎管理

## 功能邏輯

本版只調整導覽層級與顯示文案，沒有改資料表、點數、抽獎、LINE、QR Code、核銷或活動抽獎邏輯。
